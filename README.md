# VALOR — Valuation Analytics for League Optimized Rating

VALOR is a transparent, analytically rigorous player rating system for football.

Current scope: **Ligue 1 2024-2025 (MVP)**

## What makes VALOR different

- **Methodological transparency**: every choice is documented and discussable
- **Tactical grid by position**: 11 distinct weighting grids (from CB to ST)
- **Context adjustment**: strength-of-schedule via team xG (alpha=0.5)
- **Open source**: this codebase is the entire product

## Status

MVP — Ligue 1 2024-25 only. V1.5 (multi-saisons + SoS individuel) on the roadmap.
